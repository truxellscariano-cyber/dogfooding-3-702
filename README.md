# 3D 太阳系模拟器

一个使用 Three.js 构建的交互式 3D 太阳系模拟器，采用面向对象的设计模式。

## 功能特性

### 天体系统
- 🌞 **太阳**：中心恒星，带有发光效果
- 🌍 **地球**：围绕太阳公转，包含月球卫星
- 🌙 **月球**：围绕地球公转
- ⚫ **水星**：离太阳最近的行星
- 🟡 **金星**：太阳系最热的行星
- 🔴 **火星**：红色星球
- 🟤 **木星**：太阳系最大的行星
- 🪐 **土星**：拥有美丽光环的行星

### 交互功能
- **鼠标拖拽**：旋转视角
- **鼠标滚轮**：缩放场景
- **点击天体**：显示天体详细信息
- **平滑动画**：60fps 流畅渲染

### 视觉效果
- 星空背景（10000颗星星）
- 半透明轨道线
- 天体自转和公转动画
- 太阳发光效果
- 土星光环

## 技术栈

- **Three.js** 0.160.0 - 3D 渲染引擎
- **Vite** 5.0.0 - 构建工具
- **JavaScript** ES6+ - 面向对象设计

## 项目特点

### 面向对象设计
- 使用 ES6 Class 封装太阳系逻辑
- 模块化的代码结构
- 清晰的职责分离

### 性能优化
- 使用 BufferGeometry 处理星空粒子
- 优化的渲染循环
- 合理的几何体细分

## 代码架构

### SolarSystem 类
主类负责管理整个太阳系模拟器：

- `createScene()` - 创建场景
- `createCamera()` - 创建相机
- `createRenderer()` - 创建渲染器
- `createControls()` - 创建轨道控制器
- `createStarfield()` - 创建星空背景
- `createLights()` - 创建光照系统
- `createSolarSystem()` - 创建太阳系天体
- `createOrbits()` - 创建轨道线
- `animate()` - 动画循环

### 天体配置
每个天体都有独立的配置对象，包含：
- 名称、描述、详细信息
- 颜色、大小
- 轨道半径、公转速度、自转速度
